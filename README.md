# Portfolio Website

## Overview
This is my personal portfolio website showcasing my projects, skills, and experience. It serves as an online resume and portfolio to highlight my work in web development, data science, and machine learning.

## Live Demo
You can check out my portfolio website here: [Portfolio Website](https://renuka-pulavarthi.netlify.app/)

## Features
- **Responsive Design** – Optimized for all screen sizes.
- **Dark/Light Mode Toggle** – Users can switch between themes.
- **Projects Showcase** – Includes VOICEIF, House Price Prediction, and AWS Infrastructure.
- **Smooth Animations** – Enhances user experience.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Deployment**: Netlify
